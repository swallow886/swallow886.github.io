ip:132.187.39.200/24

ip地址是由32位二进制组成的，分成4个字节，用.分开。

每个字节8位，对应十进制最高255，最低是零。

2^8=255，(11111111)B=255

ip后面的/24代表的意思是:

24表示：前24位二进制数为1,剩余的**8**(32-24)位为0

这个/24的用处就是快速知道子网掩码是255.255.255.0

11111111 11111111 11111111 00000000
255 255 255 0
如何看两个ip是不是同一网段

例子：132.187.39.200/30

ip 10000100.10111011.00100111.11001000
子网掩码 11111111.11111111.11111111.11111100

子网掩码为1的部分为网络号，0的部分是主机号。

在用一网段下，

子网掩码为1的部分对应ip地址的部分不能变，只能变子网掩码为0的部分对应ip地址的部分

比如上面的，子网掩码最后两位才是0，所以对应的ip只能改变最后两位二进制数。

最后两位二进制数分别是2和1，所以要么00，01，10，11.这四种情况

对应的是0，1，2，3.

加上原来的200

对应200，201，202，203.

132.187.39.200

132.187.39.201

132.187.39.202

132.187.39.203

最后得出这四个是同一网段的。

广播地址

就是把主机号部分对应的ip地址的部分，把他变成1就是广播地址

11001000

变成1，就是

11001011

最后两位变成1，就是加上了3

广播地址就是

132.187.39.203

主机范围就是200到203

个数是：2的二次方
